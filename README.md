🏏 IPL Analysis Dashboard (2008–2025)
📌 Project Overview

This project is a complete IPL Analysis Dashboard built using Power BI to analyze Indian Premier League data from 2008 to 2025.

The dashboard provides interactive visualizations and advanced cricket insights including:

Team performance analysis
Player statistics
Season-wise records
Orange Cap & Purple Cap winners
Match trends and KPIs

This project helps users explore IPL history through dynamic dashboards and data storytelling.

🎯 Project Objectives
Analyze IPL matches and season trends
Compare team performances across years
Identify top-performing batsmen and bowlers
Analyze boundary statistics (4s & 6s)
Track season winners and runners-up
Create interactive KPIs and visual insights
🛠️ Tools & Technologies Used
Power BI
Power Query
DAX
CSV Files
📂 Dataset Information

This project uses 4 datasets:

File Name	Description
BALL_BY_BALL_DATA.csv	Ball-by-ball IPL match data
PLAYER_DATA_UPDATED.csv	Player information and details
PLAY_MATCHES_DATA.csv	Match-level IPL data
TEAM_DATA.csv	Team and franchise details
